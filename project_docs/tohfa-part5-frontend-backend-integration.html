GLOBAL INTEGRATION SPECIFICATION — READ BEFORE ANY TASK
================================================================
App name:     Tohfa
Backend:      Node.js + SQLite (better-sqlite3)
Frontend:     React (Vite or CRA) — separate folder from backend
Auth:         JWT access tokens (15-min expiry) + refresh tokens
              access_token  → stored in sessionStorage as "tohfa_access_token"
              refresh_token → stored in sessionStorage as "tohfa_refresh_token"

API BASE URL: /api   (all routes prefixed with /api)
Auth header:  Authorization: Bearer [access_token]

TOKEN REFRESH RULE:
  On every 401 response from any API call:
    1. Attempt POST /api/auth/refresh with the refresh_token from sessionStorage
    2. If refresh succeeds → update access_token in sessionStorage → retry the original request ONCE
    3. If refresh fails → clear sessionStorage → redirect to /login
  This must be implemented as an Axios/fetch interceptor (not per-screen).

CURRENCY:
  Backend stores all amounts in PAISE (integer).
  Frontend must display as ₹ with 2 decimal places.
  Utility function: formatPaise(paise) → "₹1,850.00"
  This function must be placed in src/utils/currency.js and imported where needed.

ERROR RESPONSE FORMAT (from backend):
  { "error": true, "message": "...", "code": "ERROR_CODE" }

SUCCESS RESPONSE FORMAT (from backend):
  { "success": true, "data": { ... } }

LOADING STATE RULE:
  Every API call must have 3 states rendered in the UI:
    1. Loading  — show skeleton or spinner
    2. Success  — show the data
    3. Error    — show inline error message + retry button where appropriate

RAZORPAY INTEGRATION:
  Payment flow is two-step:
    Step 1: POST /api/payments/initiate → returns { razorpay_order_id, amount_paise, key_id }
    Step 2: Open Razorpay checkout widget with those values
    Step 3: On success callback → POST /api/payments/verify with razorpay_payment_id, razorpay_order_id, razorpay_signature
  The Razorpay SDK script is loaded from: https://checkout.razorpay.com/v1/checkout.js
  The RAZORPAY_KEY_ID is exposed to the frontend via the /api/payments/initiate response (never hardcode it).

FILE UPLOADS:
  All file uploads use multipart/form-data (FormData), NOT JSON.
  Set Content-Type header to "multipart/form-data" (or let browser set it with FormData boundary).

CORS:
  Backend must allow requests from the frontend dev origin (e.g. http://localhost:5173 or http://localhost:3000).
  In production, CORS origin must match the deployed frontend URL.

PAGINATION:
  All feed/list screens use cursor-based pagination:
    Request:  ?cursor=[last_id]&limit=20
    Response: { data: [...], next_cursor: "..." | null, has_more: boolean }
  Implement infinite scroll or "Load more" button on: Home Feed, Category Products,
  Reels Feed, Order History, Seller Orders, Seller Reviews, Admin logs.

TASK 07 — AUTH: Implement global token refresh interceptor

TYPE: FRONTEND WIRING
FILE: Create FRONTEND_DIR/src/utils/apiClient.js

CREATE the Axios instance + interceptor as the single HTTP client for the entire app.
ALL other tasks must import this apiClient instead of using raw fetch or a separate axios instance.

CODE:
  import axios from 'axios';

  const apiClient = axios.create({
    baseURL: '/api',
    headers: { 'Content-Type': 'application/json' }
  });

  // Attach access token to every request
  apiClient.interceptors.request.use((config) => {
    const token = sessionStorage.getItem('tohfa_access_token');
    if (token) config.headers.Authorization = `Bearer ${token}`;
    return config;
  });

  // Auto-refresh on 401
  let isRefreshing = false;
  let failedQueue = [];

  const processQueue = (error, token = null) => {
    failedQueue.forEach(prom => error ? prom.reject(error) : prom.resolve(token));
    failedQueue = [];
  };

  apiClient.interceptors.response.use(
    (response) => response,
    async (error) => {
      const originalRequest = error.config;
      if (error.response?.status === 401 && !originalRequest._retry) {
        if (isRefreshing) {
          return new Promise((resolve, reject) => {
            failedQueue.push({ resolve, reject });
          }).then(token => {
            originalRequest.headers.Authorization = `Bearer ${token}`;
            return apiClient(originalRequest);
          });
        }
        originalRequest._retry = true;
        isRefreshing = true;
        const refreshToken = sessionStorage.getItem('tohfa_refresh_token');
        try {
          const { data } = await axios.post('/api/auth/refresh', { refresh_token: refreshToken });
          sessionStorage.setItem('tohfa_access_token', data.data.access_token);
          processQueue(null, data.data.access_token);
          originalRequest.headers.Authorization = `Bearer ${data.data.access_token}`;
          return apiClient(originalRequest);
        } catch (refreshError) {
          processQueue(refreshError, null);
          sessionStorage.clear();
          window.location.href = '/login';
          return Promise.reject(refreshError);
        } finally {
          isRefreshing = false;
        }
      }
      return Promise.reject(error);
    }
  );

  export default apiClient;

ALSO CREATE FRONTEND_DIR/src/utils/currency.js:
  export function formatPaise(paise) {
    if (paise == null) return '₹0.00';
    const rupees = paise / 100;
    return '₹' + rupees.toLocaleString('en-IN', { minimumFractionDigits: 2, maximumFractionDigits: 2 });
  }
